# Shoot Tracker Pro

This is a shooting log application built with React and Vite. You can run it locally or deploy it to GitHub Pages.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The production build will be generated in the `dist` folder.

## Deploying to GitHub Pages

1. Push this repository to GitHub.
2. Enable GitHub Pages in the repository settings using the `dist` folder as the site source.
3. Every time you run `npm run build` and push the `dist` folder to the `gh-pages` branch, your site will update.
